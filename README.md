# Local LLM App using LangChain and Ollama

## Overview

This project demonstrates how to build a local Large Language Model (LLM) application using:

- LangChain
- Ollama (local model runtime)
- Streamlit

The application allows users to send queries to a locally running open-source model without relying on external cloud APIs.
